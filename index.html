Project
fhe-language-binding
Fhe-Language-Binding is developed in Rust for binding tfhe-rs to Python, Node.js, and ES6. The binding allows users to run fhe program in Python and Javascript. The core logic of fhe computation are defined in the fhe_core module.
Project
fhe-sql
FHE-SQL is a privacy-preserving database system that enables secure querying of encrypted data using Fully Homomorphic Encryption (FHE). The system provides privacy guarantees where the server can process encrypted queries without learning the query contents or the underlying data, while clients can retrieve only the data they're authorized to access.
